📋 Overview

VBA NI is a professional invoicing and business management system built entirely in Microsoft Excel VBA. Designed for small businesses, freelancers, and entrepreneurs.

✨ Features

📄 Invoicing

  • Create professional invoices in seconds
  • PDF export with one click
  • Email sending directly from Excel
  • Multi-currency support (EUR, USD, GBP, CHF, etc.)

🇭🇷 Fiscalization 2.0 (Croatia)

  • Full compliance with Croatian tax regulations
  • JIR (Jedinstveni Identifikator Računa) generation
  • ZKI (Zaštitni Kod Izdavatelja) calculation
  • Real-time fiscalization dashboard

🌍 Multi-Language Support

14 languages included: HR, EN, DE, ES, FR, IT, PL, TR, SW, RU, CH, US, MEX, EN-EUR

💳 Payment Features

  • HUB-3 Barcode — Croatian payment standard (PDF417)
  • Swiss QR-Rechnung — Swiss payment QR code
  • Global credit card payment links

💻 System Requirements

  • Microsoft Excel: 2016, 2019, 2021, or Microsoft 365
  • Operating System: Windows 10/11 or macOS
  • Macros: Must be enabled

🚀 Installation

  1. Download VBA_NI_Lite_v1.0.xlsm
  2. Open the .xlsm file with Microsoft Excel
  3. Enable macros when prompted
  4. Go to MyInitData sheet and fill in your company info
  5. Start invoicing!
