## 🚀 Installation

### Quick Start

1. **Download** `VBA_NI_Lite_v1.0.xlsm` (or Pro version)
2. **Extract** the ZIP file to a folder
3. **Open** the .xlsm file with Microsoft Excel
4. **Enable macros** when prompted
5. **Start invoicing!**

### First-Time Setup

1. Go to **Sheet: MyInitData**
2. Fill in your company information:
   - Company name
   - Address, City, ZIP Code
   - VAT ID (OIB)
   - Bank details (IBAN, SWIFT)
   - Contact information
3. Save the file
4. You're ready to create your first invoice!

## 📖 Quick Guide

### Creating an Invoice

1. Click **"New Invoice"** button
2. Select or enter customer details
3. Add invoice items (description, quantity, price)
4. Click **"Save"**
5. Export to PDF or send via email

### Fiscalization (Pro only)

1. Create and save an invoice
2. Click **"Fiscalize"** button
3. Invoice receives JIR and ZKI codes
4. Fiscalized invoices are protected from modification

### Generating HUB-3 Barcode

1. Open an invoice
2. Click **"Generate HUB-3"**
3. Barcode appears on the invoice
4. Customer can scan for easy payment
